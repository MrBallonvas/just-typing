# Just-typing project

## About project

Typing trainer with some features:

- useful interface
- timer (15, 30, 60, 120, custom)
- words (words with optional punctuations ans numbers)
- custom text
- trainer for each finger training
- useful stats page

## Tech stack

- Frontend:
- - TypeScript, CSS, Webpack, Redux
- Backend:
- - TypeScript, Fastify
- Testing:
- - Jest
